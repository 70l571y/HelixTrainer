=                        ВВЕДЕНИЕ                              =
=================================================================
 Добро пожаловать в редактор Helix! Helix отличается от привычных
 вам редакторов тем, что он модальный — у него есть разные режимы
 работы с текстом. Основные режимы — это Normal (обычный) и Insert
 (вставка). В режиме Normal нажатые клавиши не вводят текст, а
 выполняют действия с ним. Это делает редактирование более
 эффективным. В этом учебнике вы научитесь использовать
 модальные возможности Helix. Чтобы начать, убедитесь, что
 CapsLock выключен, и удерживайте клавишу j, пока не дойдёте
 до первого урока.
=================================================================
=                  1.1 БАЗОВОЕ ДВИЖЕНИЕ КУРСОРА                 =
=================================================================
          ↑
          k       * h — влево
      ← h   l →   * l — вправо
          j       * j похожа на стрелку вниз
          ↓
 Курсор можно перемещать с помощью клавиш h, j, k, l, как показано
 выше. Клавиши-стрелки тоже работают, но hjkl быстрее, так как они
 ближе к центру клавиатуры. Попрактикуйтесь в перемещении курсора.
 Когда будете готовы, удерживайте j, чтобы перейти к следующему
 уроку.
=================================================================
=                      1.2 ВЫХОД ИЗ HELIX                     =
=================================================================
 1. Нажмите :, чтобы войти в командный режим. Курсор переместится
    вниз экрана.
 2. Введите q или quit и нажмите Enter, чтобы выйти.
 Примечание: команда quit завершится ошибкой, если есть
             несохранённые изменения. Чтобы выйти и ОТМЕНИТЬ
             изменения, введите q! или quit!.
             Как сохранять файлы, вы узнаете позже.
 Чтобы выйти из командного режима без выполнения команды,
 нажмите Escape.
 Переходите к следующему уроку.
=================================================================
=                         1.3 УДАЛЕНИЕ                        =
=================================================================
 Нажмите d, чтобы удалить символ под курсором.
 1. Переместите курсор на строку, помеченную '-->'.
 2. Переместитесь к каждому лишнему символу и нажмите d,
    чтобы удалить его.
 --> Thhiss senttencee haass exxtra charracterss.
     This sentence has extra characters.
 Как только предложение станет правильным, переходите дальше.
=================================================================
=                       1.4 РЕЖИМ ВСТАВКИ                     =
=================================================================
 Нажмите i, чтобы перейти в режим вставки.
 1. Переместитесь на строку '-->'.
 2. Переместитесь к месту, где не хватает текста, и нажмите i.
    Теперь вводимые вами символы будут отображаться в тексте.
 3. Введите недостающий текст.
 4. Нажмите Escape, чтобы выйти из режима вставки и вернуться
    в Normal mode.
 5. Повторяйте, пока строка не совпадёт с нижней.
 --> Th stce misg so.
     This sentence is missing some text.
 Примечание: в строке состояния отображается текущий режим.
             Обратите внимание: после нажатия i 'NOR' меняется на 'INS'.
=================================================================
=                      1.5 СОХРАНЕНИЕ ФАЙЛА                   =
=================================================================
 Введите :w или :write, чтобы сохранить файл.
 1. Выйдите из Helix с помощью :q!, как описано выше,
    или откройте новый терминал.
 2. Откройте файл в Helix: hx ИМЯ_ФАЙЛА
 3. Внесите изменения.
 4. Нажмите :, чтобы войти в командный режим.
 5. Введите w или write и нажмите Enter, чтобы сохранить.
 Также можно использовать wq или write-quit, чтобы сохранить
 и выйти.
 Примечание: после w можно указать путь — тогда файл сохранится
             по заданному пути.
 Примечание: если есть несохранённые изменения, в строке состояния
             рядом с именем файла появится плюс [+].

=================================================================

 * Нажмите :, чтобы войти в командный режим:
   * q / quit — выйти (ошибка при несохранённых изменениях).
   * q! / quit! — выйти и отменить изменения.
   * w / write — сохранить файл.
   * wq / write-quit — сохранить и выйти.
 * Нажмите d, чтобы удалить символ под курсором.
 * Нажмите i, чтобы войти в режим вставки; Escape — вернуться

=                  2.1 ДОПОЛНИТЕЛЬНЫЕ КОМАНДЫ ВСТАВКИ          =
=================================================================
 Как вы уже знаете, i вставляет текст перед курсором. Есть и другие
 команды вставки:
   i — вставить перед курсором.
   a — вставить после курсора (append).
   I — вставить в начало строки (первый непробельный символ).
   A — вставить в конец строки.
 1. Переместитесь на строку '-->'.
 2. Нажмите A (Shift+a), курсор переместится в конец строки,
    и вы сможете печатать.
 3. Введите недостающий текст, чтобы строка совпала с нижней.
 --> This sentence is miss
     This sentence is missing some text.
=================================================================
=                      2.2 ДОБАВЛЕНИЕ СТРОК                   =
=================================================================
 Нажмите o, чтобы вставить новую строку ниже курсора и перейти
 в режим вставки.
 Нажмите O (заглавная), чтобы вставить строку выше курсора.
 1. Переместитесь на строку '-->'.
 2. Нажмите o и введите свой ответ.
 --> What is the best editor?
=================================================================
=                        ИТОГИ ГЛАВЫ 2                       =
=================================================================
 * a — вставить после курсора.
 * I — вставить в начало строки (первый непробельный символ).
 * A — вставить в конец строки.
 * o — вставить строку ниже курсора.
 * O — вставить строку выше курсора.
=================================================================
=                  3.1 ДВИЖЕНИЯ И ВЫДЕЛЕНИЕ                   =
=================================================================
 Нажмите w, чтобы выделить от курсора до начала следующего слова.
 На самом деле, d удаляет всё выделенное. Курсор — это как бы
 выделение из одного символа.
 1. Переместитесь на строку '-->'.
 2. Переместитесь к началу лишнего слова.
 3. Нажмите w, чтобы выделить до начала следующего слова.
 4. Нажмите d, чтобы удалить выделенное.
 5. Повторяйте, пока не уберёте все лишние слова.
 --> This sentence pencil has vacuum extra words in the it.
     This sentence has extra words in it.
=================================================================
=                     3.2 ДРУГИЕ ДВИЖЕНИЯ                    =
=================================================================
 w перемещает курсор вперёд до начала следующего слова, выделяя
 пройденный текст. Это удобно как для перемещения, так и для работы
 с текстом.
 Основные движения:
   w — вперёд к началу следующего слова.
   e — вперёд к концу текущего слова.
   b — назад к началу текущего слова.
 Чтобы выделить всё слово, используйте e и b.
 1. Переместитесь на строку '-->'.
 2. Переместитесь на букву 'd'.
 3. Нажмите e, чтобы выделить часть слова.
 4. Нажмите b, чтобы выделить всё слово.
--> The Middle Kingdom.
=================================================================
=                     3.3 СЛОВА И WORDS                      =
=================================================================
 Команды w, e, b также имеют заглавные аналоги — W, E, B.
 Они работают с WORDS (слова, разделённые только пробелами),
 тогда как строчные команды также учитывают знаки препинания.
 1. Переместитесь в начало строки '-->'.
 2. Нажимайте w, чтобы по одному слову дойти до конца строки.
    Обратите внимание: 'one-of-a-kind' требует 7 нажатий,
    а '"modal"' — 3.
 3. Вернитесь в начало строки.
 4. Нажимайте W — теперь 'one-of-a-kind' и '"modal"' выделяются
    за одно нажатие.
--> Helix is a one-of-a-kind "modal" text editor
=================================================================
=                    3.4 КОМАНДА ЗАМЕНЫ (CHANGE)              =
=================================================================
 Нажмите c, чтобы заменить выделенное: это удалит выделение
 и переведёт в режим вставки.
 1. Переместитесь на строку '-->'.
 2. Переместитесь к началу неправильного слова и нажмите e,
    чтобы выделить его.
 3. Нажмите c, удалите слово и введите правильное.
 4. Повторяйте, пока строка не совпадёт с образцом.
 --> This paper has heavy words behind it.
     This sentence has incorrect words in it.
=================================================================
=                   3.5 ЧИСЛОВЫЕ ПРЕФИКСЫ                    =
=================================================================
 Введите число перед движением, чтобы повторить его нужное
 количество раз.
 1. Переместитесь на строку '-->'.
 2. Нажмите 2w — переместитесь на 2 слова вперёд.
 3. Нажмите 3e — переместитесь к концу третьего слова вперёд.
 4. Нажмите 2b — переместитесь на 2 слова назад.
 --> This is just a line with words you can move around in.
=================================================================
=                    3.6 РЕЖИМ ВЫДЕЛЕНИЯ                     =
=================================================================
 Нажмите v, чтобы войти в режим выделения (Select mode).
 Повторное нажатие v или Escape — выйти из режима.
 В этом режиме каждое движение расширяет выделение.
 1. Переместитесь на строку '-->'.
 2. Переместитесь на F в слове FOO и нажмите v2w,
    чтобы выделить два слова.
 3. Нажмите d, чтобы удалить их.
 4. Повторите с BAZ.
 --> Remove the FOO BAR distracting words BAZ BIZ from this line.
=================================================================
=                     3.7 ВЫДЕЛЕНИЕ СТРОК                    =
=================================================================
 Нажмите x, чтобы выделить всю строку. Повторное x — выделить
 следующую строку.
 1. Переместитесь на вторую строку '-->'.
 2. Нажмите x, затем d — удалить строку.
 3. Переместитесь на четвёртую строку.
 4. Нажмите x дважды или 2x, затем d — удалить две строки.
 --> 1) Roses are red,
 --> 2) Mud is fun,
 --> 3) Violets are blue,
 --> 4) I have a car,
 --> 5) Clocks tell time,
 --> 6) Sugar is sweet,
 --> 7) And so are you.
 Примечание: X работает аналогично, но не выделяет следующие строки.
             На пустой строке X ничего не делает.
=================================================================
=                   3.8 СБРОС ВЫДЕЛЕНИЯ                     =
=================================================================
 Нажмите ;, чтобы свернуть выделения до одного курсора.
 Это полезно, если вы хотите сбросить выделение без движения.
 1. Переместитесь на строку '-->'.
 2. Используйте движения, чтобы выделить текст, затем нажмите ;
    — выделение исчезнет.
 --> This is an error-free line with words to move around in.
 Примечание: Alt-; меняет направление выделения (меняет местами
             якорь и курсор).
=================================================================
=                        ИТОГИ ГЛАВЫ 3                       =
=================================================================
 * w — до начала следующего слова, e — до конца текущего,
   b — назад к началу.
 * Заглавные W, E, B работают с WORDS (только пробелы).
 * d — удалить выделенное; c — удалить и перейти в Insert mode.
 * Число перед движением повторяет его N раз.
 * v — Select mode: движения расширяют выделение.
 * x — выделить строку; повторное x — следующую строку.
 * ; — сбросить выделение до курсора.
=================================================================
=                         4.1 ОТМЕНА И ПОВТОР                =
=================================================================
 Нажмите u, чтобы отменить последнее действие.
 Нажмите U (Shift+u), чтобы повторить (redo).
 1. Переместитесь на строку '-->'.
 2. Удалите ошибку (d).
 3. Нажмите u — отменить удаление.
 4. Исправьте все ошибки.
 5. Нажмите u несколько раз — отмените исправления.
 6. Нажмите U — повторите их.
 --> Fiix the errors on thhis line and reeplace them witth undo.
=================================================================
=                 4.2 КОПИРОВАНИЕ И ВСТАВКА                  =
=================================================================
 y — скопировать (yank) выделенное.
 p — вставить после курсора.
 P — вставить перед курсором.
 1. Переместитесь на строку '-->', курсор на "b" в banana.
 2. Нажмите w (выделить "banana"), затем y — скопировать.
 3. Переместитесь между "2" и "3", нажмите p — вставить.
 4. Повторите между "3" и "4".
 --> 1 banana 2 3 4
     1 banana 2 banana 3 banana 4
 Примечание: при удалении или замене текст автоматически
             копируется. Используйте Alt-d / Alt-c, чтобы этого
             избежать.
 Примечание: Helix не использует системный буфер обмена по умолчанию.
             Используйте Пробел+y / Пробел+p для копирования/
             вставки в системный буфер.
=================================================================
=                     4.3 ПОИСК В ФАЙЛЕ                     =
=================================================================
 / — поиск вперёд; Enter — подтвердить.
 n — следующее совпадение.
 N — предыдущее совпадение.
 1. Нажмите /, введите слово, например 'banana'.
 2. Нажмите Enter.
 3. Используйте n/N для перехода по совпадениям.
 Поиск использует регулярные выражения.
 Примечание: ? (Shift+/) — поиск назад.
             В отличие от Vim, направление не меняется:
             n всегда вперёд, N — назад.
=================================================================
=                        ИТОГИ ГЛАВЫ 4                       =
=================================================================
 * u — отменить, U — повторить.
 * y — копировать, p — вставить после, P — до.
   * Пробел+y/p — работа с системным буфером.
 * / — поиск вперёд, ? — назад.
   * n/N — переход по результатам поиска.
=================================================================
=                     5.1 НЕСКОЛЬКО КУРСОРОВ                =
=================================================================
 C — дублировать курсор на следующей подходящей строке.
 1. Переместитесь на первую строку '-->', курсор после '-->'.
 2. Нажмите C — появится второй курсор.
    Лишняя строка пропускается.
 3. Исправьте обе строки в Insert mode — изменения применятся сразу.
 4. Нажмите , — убрать первый курсор.
 --> Fix th two nes at same ime.
 -->
 --> Fix th two nes at same ime.
     Fix these two lines at the same time.
 Примечание: Alt-C — то же самое, но вверх.
=================================================================
=                    5.2 КОМАНДА ВЫБОРА (SELECT)             =
=================================================================
 s — выделить все совпадения шаблона в текущем выделении.
 1. Переместитесь на строку '-->'.
 2. Нажмите x (выделить строку).
 3. Нажмите s, появится запрос.
 4. Введите 'apples', Enter — оба вхождения выделены.
 5. Нажмите c и замените на 'oranges'.
 6. Escape — выйти из Insert mode.
 7. Нажмите , — убрать второй курсор.
 --> I like to eat apples since my favorite fruit is apples.
     I like to eat oranges since my favorite fruit is oranges.
=================================================================
=                    5.3 ВЫБОР ПО РЕГУЛЯРНОМУ ВЫРАЖЕНИЮ      =
=================================================================
 Команда s поддерживает регулярные выражения.
 1. Переместитесь на строку '-->'.
 2. Выделите строку (x), затем нажмите s.
 3. Введите '  +' (два и более пробела), Enter.
 4. Нажмите c и замените на один пробел.
 --> This  sentence has   some      extra spaces.
     This sentence has some extra spaces.
 Примечание: для замены по всему файлу выделите его (% или несколько x),
             затем используйте s и c.
=================================================================
=                      5.4 ВЫРАВНИВАНИЕ ВЫДЕЛЕНИЙ            =
=================================================================
 & — выровнять выделения по "голове" (движущемуся концу).
 1. Переместитесь на первую строку '-->', курсор на пробел после -->.
 2. Нажмите C четыре раза (или 4C).
 3. Нажмите W, чтобы выделить числа и скобки.
 4. Нажмите & — выравнивание.
 --> 97) lorem
 --> 98) ipsum
 --> 99) dolor
 --> 100) sit
 --> 101) amet
=================================================================
=                 5.5 РАЗДЕЛЕНИЕ ВЫДЕЛЕНИЙ НА СТРОКИ         =
=================================================================
 Alt-s — разделить выделения по строкам.
 1. Переместитесь на первую строку таблицы.
 2. Выделите всю таблицу (6x).
 3. Нажмите Alt-s — каждая строка станет отдельным выделением.
 4. Нажмите & — выровнять.
    | FRUIT   | AMOUNT |
    |---------|--------|
 | Apples  | 8      |
    | Bananas | 6      |
  | Oranges | 3      |
     | Donuts  | 4      |
=================================================================
=                        ИТОГИ ГЛАВЫ 5                       =
=================================================================
 * C — добавить курсор на следующей строке; Alt-C — на предыдущей.
 * s — выделить все вхождения регулярного выражения.
 * & — выровнять выделения.
 * Alt-s — разделить выделения по строкам.
=================================================================
=                 6.1 ВЫДЕЛЕНИЕ ДО СИМВОЛА                   =
=================================================================
 f<символ> — выделить до и включая символ.
 t<символ> — выделить до, но не включая символ.
 F/T — то же самое, но назад.
 1. Переместитесь на строку '-->', курсор на первый дефис.
 2. Нажмите f[ — выделит до квадратной скобки.
 3. Нажмите d — удалить.
 4. Перейдите в конец строки и повторите с F].
 5. На второй строке используйте t/T, чтобы убрать дефисы.
 --> -----[Free this sentence of its brackets!]-----
 --> ------Free this sentence of its dashes!------
 Примечание: в отличие от Vim, поиск не ограничен строкой —
             ищет по всему файлу.
=================================================================
=                    6.2 КОМАНДА ЗАМЕНЫ СИМВОЛОВ            =
=================================================================
 r<символ> — заменить все выделенные символы на <символ>.
 1. Переместитесь на вторую строку таблицы, курсор на первый =.
 2. Нажмите t| (Shift+\), чтобы выделить до вертикальной черты.
 3. Нажмите r- — заменить на дефисы.
 | Month | Days |
 |=======|------|
 | Jan   | 31   |
 | Feb   | 28   |
 | Mar   | 31   |
 | ...   | ...  |
=================================================================
=                         6.3 ПОВТОРЕНИЕ                    =
=================================================================
 . — повторить последнюю команду вставки.
 Alt-. — повторить последнее выделение с f/t.
 1. Переместитесь на строку '-->'.
 2. Сделайте изменение и повторите его с помощью '.'.
 3. Попробуйте Alt-. с f/t для выделения нескольких фрагментов.
 --> This is some text for you to repeat things. You can repeat
     insertions like changing words, or repeat selections like
     f / t.
=================================================================
=                        ИТОГИ ГЛАВЫ 6                       =
=================================================================
 * f/F — выделить до и включая символ.
 * t/T — выделить до, но не включая символ.
 * r — заменить выделенные символы.
 * . — повторить последнюю вставку.
 * Alt-. — повторить последнее f/t-выделение.
=================================================================
=                  7.1 ЗАМЕНА НА СКОПИРОВАННЫЙ ТЕКСТ         =
=================================================================
 R — заменить выделенное ранее скопированным текстом.
 1. Переместитесь на строку '-->'.
 2. Нажмите w, чтобы выделить "watermelons", затем y — скопировать.
 3. Выделите "oranges" с помощью w.
 4. Нажмите R — заменить на "watermelons".
 --> I like watermelons because oranges are refreshing.
     I like watermelons because watermelons are refreshing.

is spilling over
onto other
lines.

 3. Повторите для числа 3 — станет 4.

=                         8.1 РЕГИСТРЫ                      =
=================================================================
 Регистры — контейнеры, идентифицируемые символами, для хранения
 текста, поисковых запросов, макросов и т.д.
 "<символ> — выбрать регистр <символ>.
 1. Переместитесь на строку '-->'.
 2. Выделите "watermelons", y — скопировать (в регистр по умолчанию).
 3. Выделите "bananas".
 4. Нажмите "b y — скопировать в регистр b.
 5. Выделите "mangoes", нажмите R — заменит на "watermelons".
 6. Выделите "pineapples", нажмите "b R — заменит на "bananas".
 --> I like watermelons and bananas because my favorite fruits
     are mangoes and pineapples.

=                          8.2 МАКРОСЫ                      =
=================================================================
 Макросы — запись последовательности действий.
 Q — начать запись макроса (по умолчанию в регистр @).
 Повторное Q — остановить запись.
 q — выполнить макрос из регистра @.
 1. Переместитесь на первую строку '-->', курсор на '>' в -->.
 2. Нажмите Q — начать запись.
 3. Исправьте строку, как в образце.
 4. Нажмите Q — остановить запись.
 5. Переместитесь на следующую строку, курсор на '>'.
 6. Нажмите q — применить макрос.
 --> ... sentence doesn't have its first and last ... .
 --> ... sentence doesn't have its first and last ... .
     This sentence doesn't have its first and last word.

=                        ИТОГИ ГЛАВЫ 8                       =
=================================================================
 * " — выбрать регистр.
 * Q — записать макрос (в регистр по умолчанию @).
 * q — выполнить макрос из @ или выбранного регистра.

=                  9.1 ПОИСК ПО ВЫДЕЛЕНИЮ                    =
=================================================================
 Последний поиск с помощью / хранится в регистре /.
 n и N используют этот регистр.
 * — скопировать выделенное в регистр /, сделав его шаблоном поиска.
 1. Переместитесь на строку '-->'.
 2. Выделите "horse" (e), нажмите *.
 3. Используйте n/N для перехода по вхождениям.
 --> A horse is a horse, of course, of course,
 --> And no one can talk to a horse of course.
 Примечание: * — это как "/y — просто копирует выделенное в /.
=================================================================
=           9.2 ДОБАВЛЕНИЕ ВЫДЕЛЕНИЙ ПО ПОИСКУ               =
=================================================================
 В режиме выделения (v) n и N добавляют новые выделения вместо
 перехода к ним.
 1. Переместитесь на строку '-->'.
 2. Выделите первый "bat", нажмите *.
 3. Нажмите v — режим выделения.
 4. Нажмите n — выделится второй "bat".
 5. Используйте c или r, чтобы заменить на "cat".
 --> Everybody wants to be a bat,
 --> because a cat's the only bat
 --> who knows where it's at.
=================================================================
=                     9.3 ИСПОЛЬЗОВАНИЕ JUMPLIST             =
=================================================================
 Helix запоминает "прыжки" (большие перемещения) в jumplist.
 Ctrl-s — сохранить текущую позицию в jumplist.
 Ctrl-i — вперёд по jumplist ("in").
 Ctrl-o — назад по jumplist ("out").
 1. Нажмите Ctrl-s.
 2. Переместитесь далеко.
 3. Нажмите Ctrl-o — вернётесь туда, где сохранили.
=================================================================
=             9.4 ПЕРЕХОД ПО ДВУХСИМВОЛЬНЫМ МЕТКАМ           =
=================================================================
 gw — включить двухсимвольные метки на начале слов.
 Нажмите любые два подсвеченные символа — перейдёте туда.
 Escape — отменить.
 1. Переместитесь в начало строки '-->'.
 2. Нажмите gw, затем два символа, заменивших "he" в "here".
 --> This is just a simple line of text.
     There may be many such lines
     But you really want to jump here!
     This is fast with the 2-character labels.

 * * — выделенное → регистр поиска.
 * v + n/N — добавить выделения по поиску.
 * Ctrl-s — сохранить позицию; Ctrl-o/i — назад/вперёд по jumplist.
 * gw — двухсимвольные метки; любой символ — перейти; Escape — отмена.
=================================================================
=             10.1 ЦИКЛИЧЕСКИЙ ПЕРЕБОР ВЫДЕЛЕНИЙ            =
=================================================================
 ) — следующее выделение становится основным.
 ( — предыдущее выделение становится основным.
 Alt-, — удалить основное выделение.
 1. Переместитесь на строку '-->'.
 2. Выделите обе строки (xx или 2x).
 3. Нажмите s, введите "would", Enter.
 4. Используйте ( и ) для выбора основного выделения,
    Alt-, — удалить второе "would".
 5. Нажмите c и введите "wood".
 --> How much would would a wouldchuck chuck
 --> if a wouldchuck could chuck would?
=================================================================
=             10.2 ЦИКЛИЧЕСКАЯ ЗАМЕНА СОДЕРЖИМОГО            =
=================================================================
 Alt-) — сдвинуть содержимое выделений вперёд.
 Alt-( — сдвинуть назад.
 1. Переместитесь на строку '-->'.
 2. Выделите обе строки (xx).
 3. Нажмите s, введите "through|water|know", Enter.
 4. Используйте Alt-( и Alt-) для циклической замены.
 --> Jumping through the water,
 --> daring to know.
=================================================================
=                     10.3 ИЗМЕНЕНИЕ РЕГИСТРА               =
=================================================================
 ~ — переключить регистр выделенных букв.
 ` — сделать строчные.
 Alt-` — сделать заглавные.
 1. Первая строка: выделите неправильные буквы, нажмите ~.
 2. Вторая строка: x, затем ` — всё в нижний регистр.
 3. Третья строка: x, затем Alt-` — всё в верхний регистр.
 --> thIs sENtencE hAs MIS-cApitalIsed leTTerS.
 --> this SENTENCE SHOULD all be in LOWERCASE.
 --> THIS sentence should ALL BE IN uppercase!
=================================================================
=                   10.4 РАЗДЕЛЕНИЕ ВЫДЕЛЕНИЙ                =
=================================================================
 S — разделить каждое выделение по регулярному выражению.
 1. Переместитесь под ---.
 2. Выделите обе строки (2x).
 3. Нажмите S, затем введите "\. |! ", Enter.
    (Обратите внимание на пробелы!)
    Это разделит выделение по точкам и восклицательным знакам.
 4. Alt-; — развернуть выделения.
 5. ; — свернуть до первых букв.
 6. Alt-` — сделать их заглавными.
---
these are sentences. some sentences don't start with uppercase
letters! that is not good grammar. you can fix this.

=================================================================
 * ) / ( — следующее/предыдущее выделение основным.
 * Alt-, — удалить основное выделение.
 * Alt-) / Alt-( — циклически сдвинуть содержимое.
 * ~ — переключить регистр; ` / Alt-` — нижний/верхний регистр.
 * S — разделить выделения по регулярному выражению.

 Ctrl-c — закомментировать строку под курсором.

 1. Переместитесь на строку '-->'.
 2. Нажмите Ctrl-c — строка закомментируется.
 3. Нажмите ещё раз — раскомментируется.
 --> Comment me please
=================================================================
=                 11.2 КОММЕНТИРОВАНИЕ НЕСКОЛЬКИХ СТРОК      =
=================================================================
 Используйте выделение или несколько курсоров, затем Ctrl-c.
 1. Переместитесь на первую строку '-->'.
 2. Выделите или добавьте курсоры на остальные строки.
 3. Нажмите Ctrl-c — все закомментируются.
 --> How many are you going to comment?
 --> Is this enough for a comment?
 --> What are you doing?!
 --> Stop commenting me!
 --> AAAAaargh!!!
 Примечание: если строка уже закомментирована, она не раскомментируется,
             а добавится ещё один уровень комментария.

 * Раскомментирование не работает автоматически — только ручное.

=                   12.1 ПЕРЕХОД К ПАРНЫМ СКОБКАМ            =
=================================================================
 m — войти в режим работы со скобками (match mode).
 mm — перейти к парной скобке.
 1. Переместитесь на строку '-->', курсор на (.
 2. Нажмите mm — перейдёте к ).
 Аналогично для [], {}.
 --> you can (jump between matching parentheses)
 --> or between matching [ square brackets ]
 --> now { you know the drill: this works with brackets too }
=================================================================
=              12.2 ВЫДЕЛЕНИЕ ВНУТРИ СКОБОК                  =
=================================================================
 mi( — выделить содержимое внутри круглых скобок (без скобок).
 Аналогично для [], {}, "", и т.д.
 1. Переместитесь на строку '-->', курсор внутри скобок.
 2. Нажмите mi( — выделится только содержимое.
 3. Можно редактировать (например, c).
 --> outside and (inside x parentheses) - and outside again
=================================================================
=               12.3 ВЫДЕЛЕНИЕ ВМЕСТЕ СО СКОБКАМИ            =
=================================================================
 ma( — выделить содержимое со скобками.
 1. Переместитесь на строку '-->', курсор внутри ().
 2. Нажмите ma( — выделится "( select x around )".
 3. Можно удалить: ma(d.
 --> you ( select x around ) to include delimiters in the select
=================================================================
=                12.4 ОБВОДКА ТЕКСТА СКОБКАМИ                =
=================================================================
 ms( — обернуть выделенное в скобки ().
 1. Выделите "select all of this" (например, v4e).
 2. Нажмите ms( — текст обернётся в ().
 Аналогично для '', "", [], {}.
 --> so, select all of this, and surround it with ()
 --> surround this WORD !
=================================================================
=              12.5 УДАЛЕНИЕ ОБВОДКИ (СКОБОК)               =
=================================================================
 md( — удалить ближайшую пару скобок вокруг курсора.
 1. Переместитесь внутрь (), например на 'x'.
 2. Нажмите md( — скобки исчезнут.
 Работает и для вложенных, и для кавычек.
 --> delete (the x pair of parentheses) from within!
 --> delete "layers "of" quote marks" too
=================================================================
=             12.6 ЗАМЕНА ОБВОДКИ (СКОБОК)                  =
=================================================================
 mr([ — заменить круглые скобки на квадратные.
 1. Переместитесь внутрь (), например на 'x'.
 2. Нажмите mr([ — () заменятся на [].
 Работает для любых пар.
 --> replace the (pair from x within), with something else
 --> some (nested surroundings [can be replaced])
 --> this "works with 'other surroundings' too"
=================================================================

=================================================================
 m — режим работы со скобками:
  * mm — перейти к парной скобке.
  * mi( — выделить внутри скобок.
  * ma( — выделить со скобками.
  * md( — удалить скобки.
  * ms( — обернуть выделенное в скобки.
  * mr([ — заменить одни скобки на другие.
